This is just a personal blog. I use a few tools to run the website, and I've listed them below.

Jekyll and Github

This website is built using Jekyll and hosted on Github Pages. In non-technical terms, this is a pretty simple setup which doesn't require any database or backend, user login, etc. - it just serves a simple HTML webpage which you're reading right now. Github kindly host these types of site for free (since they are just HTML files). That being said, they may collect some information in the background for the purposes of security and to maintain their service. Here's a link to their privacy policy if you'd like to read more: Github Privacy Policy
